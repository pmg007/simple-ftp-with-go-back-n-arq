
How to run the code?


1. Place the 1MB file to be sent in the same location as of the client.py

2. Then run the server code by navigating to its locaation on terminal and :
python server.py #server-port #file-name #probability

3. After that, run the client code as:
python client.py #server-ip #server-port #file-name #N-value #mss-value

4. After successful transmission we can see that the file gets downloaded in the same direcory as of the server.py
